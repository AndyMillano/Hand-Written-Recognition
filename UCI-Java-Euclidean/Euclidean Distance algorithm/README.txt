How to run algorithm
to run the algorithm simply load the project into Eclipse IDE,
or create a new file and place the code from EuclidianDistance.java and them load the two datasets into the project

to run the first fold test navigate to line 28 in EuclideanDistance class and make sure the line is not commented, and also make sure line 29 is commented
to run the second fold test navigate to line 29 uncomment it, and them comment line 28


the result of the first fold test was :	98.93238%
the result of the second fold test was: 99.07473%
total result average: 99.003555%
